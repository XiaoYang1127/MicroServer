# 微服务架构设计

## 微服务介绍

- 微服务是系统架构上的一种设计风格，主旨是将一个原本独立的系统拆分成多个小型服务，这些小型服务都在各自独立的进程中运行，服务之间通过特定的通信方式协作
- 被拆分成的每一个小型服务都围绕着系统中一些耦合度较高的业务功能进行构建，每个服务可独自维护自身的数据存储，业务开发，以及独立部署
